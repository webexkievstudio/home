    Из чего состоит стоимость сайта?

    Качественный сайт - это работа специалистов, которые вкладывают в разработку продукта
    весь свой профессионализм и драгоценное время

    Copywriter

    Создает основной контент, наполняет
    сайт уникальной информацией.
    Он грамотно формулирует текстовый
    посыл, ориентированный на ЦА сайта.

    UI/UX Designer

    Разрабатывает визуальную оболочку
    сайта. Вначале создает прототип сайта,
    на основе которого рисует мокап
    для дальнейшей передачи его
    Front-end разработчику.

    Front-end Developer

    Занимается версткой сайта, нарисованного
    веб-дизайнером. Проводит последующее
    тестирование продукта на различных
    платформах - от смартфонов
    до полноценных десктопов.

    Back-end Developer

    Пишет содержательную часть
    сайта - прорабатывает логику
    взаемодействий его активных
    элементов.

    Project Manager

    Контролирует процесс разработки сайта
    на всех этапах. Взаимодействует между
    участниками команды разработчиков.
    Контактирует с заказчиком от имени студии.

    QA Engineer

    Занимается тестированием сайта,
    ищет баги, передает разработчику для исправления
